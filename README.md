 SpamPanel

> Многофункциональная панель для рассылки сообщений через Telegram Bot и Discord Webhook.


---


## Возможности

- Поддержка **Telegram Bot** и **Discord Webhook** в одном приложении
- Переключение между режимами прямо внутри программы
- Очередь сообщений с настраиваемым интервалом (секунды / минуты / часы)
- Пауза и остановка спама в любой момент
- Чат-монитор входящих сообщений (Telegram)
- Ручная отправка сообщений боту
- Логи всех действий
- Профиль бота с аватаром и статистикой сессии
- Автосохранение конфига рядом с `.exe`
- Тёмный UI на базе CustomTkinter


---


## Запуск

### Готовый .exe

Скачай [SpamPanel.exe]( https://drive.google.com/file/d/1czqsSXmpCmkYTP73w9kGRgSrL8yLamjq/view?usp=sharing ) и запусти двойным кликом.

---


## Использование

**Telegram режим:**
1. Вставь Bot Token (получить у [@BotFather](https://t.me/BotFather))
2. Вставь Chat ID
3. Добавь сообщения в очередь
4. Нажми `Запустить бота` → `Начать спам`

**Discord режим:**
1. Вставь Webhook URL (Настройки канала → Интеграции → Вебхуки)
2. Опционально: имя отправителя и URL аватара
3. Добавь сообщения в очередь
4. Нажми `Начать спам`
